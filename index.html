Sobre Nós

O iDotaPet é um projeto voltado para conectar animais que precisam de um lar com pessoas dispostas a adotar. Nosso objetivo é incentivar a adoção responsável e ajudar a reduzir o número de animais abandonados.

Este site foi desenvolvido como um projeto acadêmico, utilizando tecnologias web para criar uma solução simples, acessível e com impacto social positivo.
